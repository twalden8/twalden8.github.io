Review of State Websites

Alaska

  • I do like the colors of this website but not so much the design. The colors are eye pleasing but the website just looks kind of old and out dated.
  • The website loads quickly
  • The menu is easy to navigate. Everything is very user friendly and easy to find.
  • The website pretty much stays the same in different screen sizes. If you resize the window and make it smaller, the website doesn't resize or anything, it just stays how it is.

Arkansas

  • I love this design and color. It is very pleasing to the eye. The design is super cool and is very nice while you scroll down the website
  • The website loads a little slower than usual. It loads pretty fast when you just go straight into it but when you click on links in the website its kind of slow.
  • The website is very easy to navigate. You can find items very easy. In the heading theres links that show exactly what you need.
  • The website does work very well in multiple screen sizes. Everything resizes exactly how you would want it.

Kansas

  • I do not like the color or design of this website
  • For me the website loaded pretty quickly. Theres nothing really to load except for a search bar.
  • The menu is terrible. I don't know how they thought of this but I really do not like it. All there is is a search bar so you can search for what you want. If you know exactly what you are looking for, the search bar could be helpful, but if you aren't sure exactly what it's called then you are out of luck.
  • The website works fine in all screen sizes.

Maine

  • Maine.gov has a decent color scheme. It's nothing special but at least it isn't as bad as Kansas.
  • The website loads quickly
  • The menu is very easy to navigate. It has a trending bar which I really like. It also has a search bar, and a header with links to everything you would need.
  • The website works fine in all screen sizes

Maryland

  • This color scheme is not my favorite. The red orange and yellow just don't look that great together with the lake and tree picture.
  • The website loads quickly
  • The menu is very easy to navigate. It has a header with links you will pretty much only use. You can find everything pretty easily.
  • Website works fine in all screen sizes.

Favorite State Website

My favorite state website is definitely Arkansas. The color scheme is very nice and the scrolling animation looks really cool. Everything is super easy to find and it has very cool features with it.

Least Favorite State Website

My least favorite state website is for sure Kansas. Whoever had the great idea of only a search bar should probably get fired. I'm sure it works fine, but I feel like it is not very user friendly at all. I feel like my grandma could never succeed in finding what she wants if she opened this. There isn't really a color scheme to this website. It is just a big picture in the background. It could be a good website if they just implemented some sort of menu. Final rating is a 2/10. Not the worst website I've ever seen but pretty darn close.
